## Installation

1. Clone project repositories
2. Install dependencies

- Install using yarn

```bash
    yarn && yarn upgrade
```

- Or using npm

```bash
    npm install
```

3. Run development server or build production

- Run using yarn

```bash
    // development server
    yarn dev

    // production build
    yarn build
```

- Or run using npm

```bash
    //development server
    npm run dev

    //production build
    npm run build

```
